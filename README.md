# MIPS Assembly Lab Project

## Overview
This repository contains MIPS Assembly Language lab work and coursework developed for the Computer Organization and Architecture course.

The project includes:
- MIPS Assembly programs
- Lab tasks and solutions
- Reports and documentation
- Presentation slides
